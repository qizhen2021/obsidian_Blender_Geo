# 节点
![[Pasted image 20240817201225.png|200]]

>[!目的]
>翻转面节点会反转每个面的法线法向。大家建模的时候都会碰见网格面正面和反面的情况，然后根据情况来调整正反面，跟这个节点是一样的。


# 输入
网格

	要翻转法线的网格
选中项

	具体选择要翻转的面

# 输出
网格

	翻转面的网格
