# 节点
![[IMG-20240816214342356.png|200]]

>[!目的]
>细分网格节点将每个面分割成多个面。换句话说，它与细分表面节点相同，但没有平滑部分。

# 输入
网格

	要细分的网格
级别

	要细分的次数

# 输出
网格

	细分后的网格
