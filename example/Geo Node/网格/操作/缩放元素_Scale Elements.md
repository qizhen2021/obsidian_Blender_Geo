# 节点
![[IMG-20240816214308652.png|200]]

>[!目的]
>缩放元素节点分别缩放给定几何体的每个元素。

# 输入
几何体

	要缩放的几何体。只能是网格。
选中项

	选择要缩放的几何体部分。
缩放

	决定几何体的缩放比例值。
	数值1表示原始比例。
中心
	设置一个几何体向其缩放的轴心位置。如果未选中，则选择连接顶点的平均位置作为缩放中心

# 输出
几何体

	缩放后的几何体。

# 额外设置
域类型

	选择通过缩放网格的边缘或面来缩放几何体
缩放模式

	选择均衡以相同的因子（统一）缩放每个轴的所有元素，或为每个轴设置不同的因子（单轴）。
