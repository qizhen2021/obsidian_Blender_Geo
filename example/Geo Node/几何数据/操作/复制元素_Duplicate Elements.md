# 节点
![[Pasted image 20240816160645.png|200]]

>[!目的]
>复制元素节点可以按你想要的次数复制几何体。然后，你可以使用副本编号输出插槽更改各个副本的字段。

# 输入
几何体

	在这种情况下，几何输入允许以下几何类型：网格、曲线、点、和体积。
选中项

	选择几何的哪些部分应受此节点影响。在这种情况下，选择几何的哪些部分将被复制。
数量

	您希望生成的副本数量。值为1意味着只有源几何的一个版本。

# 输出
几何体

	复制的几何体。
副本编号

	一个有序列表，包含每个重复对象的索引。
