# 节点
![[Pasted image 20240816145837.png|200]]

>[!目的]
>按距离合并节点将在给定距离内的顶点和点合并在一起。（如编辑模式按M按距离合并）

# 输入
几何体

	在这种情况下，几何体输入允许以下类型的几何体：网格、曲线和点。
选中项

	选择几何体的哪些部分应受此节点影响。
距离

	如果两点之间的距离小于此值，这些点将合并为一个点。

# 输出
几何体

	输出合并后的几何体

# 额外设置
相连项

	如果设置为相连项，则只有通过边连接的顶点才会合并在一起
