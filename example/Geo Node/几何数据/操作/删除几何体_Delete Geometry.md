# 节点
![[Pasted image 20240816140441.png|200]]

>[!目的]
>删除几何节点会删除其接收到的任何几何体。

# 输入
几何体

	在这种情况下，几何体输入允许以下类型的几何体：网格、曲线和点。
选择

	选择要删除的几何体部分。

# 输出
几何体

	删除后剩余的几何体

# 额外设置
域类型

	选择删除过程的域类型。你可以选择删除点、边、面、样条线或实例。
删除模式

	当域类型为点、边或面时，您可以选择删除与之相关的所有内容，仅删除边或面，或仅删除面。
